BAHT Capstone: Technical Direction and Tool Development

USD Assembly Pipeline

An assembly workflow that uses Universal Scene Description (USD) to handle managing assemblies and set dressing in our animation pipeline.

Turntable Automator

For any given asset, this tool automates the creation of a turntable within both modeling and surfacing (with rendering done in Arnold) for easy look dev.

Project Manager

Tracks all assets and shots within the project, allowing user to view details about each, re-publish, change version, and more.

About My Capstone

For my capstone, I explored creating tools within an animation pipeline as a technical director in UTD's Animation Lab. I worked on the above tools over the course the spring 2025 semester (with the exception of the USD assembly pipeline, which I have been continually working on for a few semesters now). Each of these tools was made with the purpose of understanding artist workflows and streamlining their tasks in some way. Click on the projects above to learn more about each individually!

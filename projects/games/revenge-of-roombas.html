SGDA Arcade Jam | The Revenge of the Roombas

This game was made for the Student Game Developer Association's arcade jam in Spring 2022, where the theme was "Combos." Our game won the award for "Best Technical Achievement."

Prior to making Revenge of the Roombas, I had no experience with scripting in C# and how it works with Unity. But over the course of the week, I programmed the UI and helped write the enemy AI for the Roombas with one of my teammates.

For the enemy AI, we created a StateMachine base class that each of the three types of enemy Roombas (exploding, shooting, and wandering/seeking alternating) inherited to create each of their specific behaviors. The base class allowed the Roombas to transition between each enemy type's states (for example, the last Roomba enemy type alternates between seeking out the player and wandering aimlessly).

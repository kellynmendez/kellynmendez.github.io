Maya Object Stacker Tool

This tool was created over the course of the Spring 2022 semester in my Scripting for Animation class as an introduction to tools development. The object stacker tool creates stacks of objects, and objects for each part of the stack are randomly selected from groups of objects specified by the user. After creating the stacks, a user can also provide an XML file with position information that will move each stack to its desired position.

This project was broken up into three parts over the course of the semester: stacking capability, draft GUI for specifying object groups, and the final GUI with the XML file capability. To read through the code, view the Github repo here.

Turntable Automator

I automated the creation of a turntable for any asset within both modeling and surfacing. The tool first reads an XML file that has pre-set settings for playblasting or rendering for the current project. Within this XML, there are two different quality options for the turntable, either low (if the artist just needs a quick creation of the turntable) or high quality.

Modeling

  1. The USD kind is set to assembly. This is done so that an assembly can be imported into another assembly in the future if needed.
  2. The base prim is set to be the default prim for the USD. All assets in the future are automatically imported into the contents prim.

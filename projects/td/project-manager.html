Project Manager

The project manager is currently a work in progress, but it is used to check the status and details of assets and shots within a project. A user is, for all assets and shots, able to view its details, publish it, version it back if needed, and more.

Use Cases

  1. Publish Current Asset: This runs a subprocess that will open the current work file for this asset or shot and publish it.
  2. Version Back: This versions the asset back a single version. In the future, I would like for the user to be able to choose what version they would like to version back to.

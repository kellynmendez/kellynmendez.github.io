Pipeline TD Work

I have worked as a TD in Animation Lab for nearly three years, and in that time I have worked on three finished animated shorts: Bad Timing (2022), Red Hat Ringo (2023), and Sunny (2024). The short I am currently working on is The Kraken's Tooth, a fun dark comedy about fish pirates searching for treasure under the sea, which is set to premiere in December 2025.

My job as technical director is to maintain our pipeline and codebase, develop tools for the artists, and provide front-line support for troubleshooting any production problems that might arise. Our production pipeline uses Maya and its Python API for all disciplines (with the exception of Red Hat Ringo, where the lighting pipeline was moved to Unreal 5) and uses Flow (ShotGrid/Shotgun) for asset and shot tracking. I have contributed many different tools and workflows to Animation Lab's pipeline since starting, some of which you can read more about below!

Maya Object Stacker Tool

A tool that allows a Maya user to stack selected objects in any order and at a specified height and separation. A user can also choose to give an XML file with position information that will move each stack to its specified position.
